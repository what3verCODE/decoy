# Введение

> 🚧 Русский перевод ещё не готов. Ниже — английский источник.

Decoy is a fast, contract-first HTTP mock you point a base URL at. You author mock **routes**,
group them into switchable **collections**, point any client's base URL at the running server,
and develop or test against deterministic scenarios — without waiting for a real backend.

See the [English Introduction](/guide/start/introduction) for the full text.
